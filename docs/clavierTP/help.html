Claviers d'accordéons diatoniques

Ce petit utilitaire permet de visualiser différents plans de clavier d'accordéons diatoniques et d'en jouer, c'est à dire de voir quelles touches correspondent à un accord ou une gamme.

Jouer avec le clavier

Il suffit de cliquer sur une touche et de se déplacer vers la droite ou vers la gauche comme pour tirer ou pousser relativement à la position du soufflet (au centre). Les touches correspondant au mode sélectionné changent alors de couleur. Cette couleur est différente selon que l'on tire ou que l'on pousse.
Le mode par défaut est accord mineur. Le mode utilisé est affiché en haut à gauche de l'écran.

Sélectionner un mode de jeu

Pour sélectionner un mode de jeu, il faut cliquer sur le mode utilisé (en haut à gauche de l'écran). Un menu apparait qui permet de changer de mode de jeu.

Sélectionner un clavier

Le plan de clavier courant est affiché en haut à gauche de l'écran. Pour en changer, il faut cliquer sur le clavier. Un menu apparait qui permet de sélectionner un plan de clavier différent.
Si votre plan de clavier n'est pas dans la liste, vous pouvez me l'envoyer et je le rajouterai.
